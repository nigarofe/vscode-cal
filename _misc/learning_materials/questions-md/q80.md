# Question Statement
Encontre uma solução particular da equação não-homogênea
$$
y'' + 3y' + 4y = 3x + 2.
$$
# LLM Solution %% fold %%
Se $P(0) \neq 0$: supõe-se $y_p = A_0 + A_1 x + \dots + A_m x^m$

$\textbf{Resposta:}$ Uma solução particular da EDO é
$$
y_p(x) = \frac{3}{4}x - \frac{1}{16}.
$$
